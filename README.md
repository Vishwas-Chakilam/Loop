# Loop

Loop is a simple and intuitive habit tracker with AI insights. Use it to build and maintain healthy habits through daily tracking, reminders, and progress visualization.

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies: `npm install`
2. Set `GEMINI_API_KEY` in [.env.local](.env.local)
3. Run the app: `npm run dev`
